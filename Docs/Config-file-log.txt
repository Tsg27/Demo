
1- Localize o Arquivo php.ini:

No terminal do VSCode, 
execute o seguinte comando abaixo para encontrar arquivo exibirá o caminho para o arquivo "php.ini". 
COMANDO: php --ini
 


2 - Abra o arquivo php.ini em um editor de texto de sua escolha, como Bloco de Notas ou VSCode.
Procure pela configuração error_log no arquivo. 

Você pode usar a funcionalidade de busca do editor para localizar rapidamente a linha.
Altere ou adicione a linha error_log para apontar para o arquivo de log desejado. Por exemplo:

error_log="C:\Xampp\htdocs\schoollibrary\logs\php_errors.log"


Após fazer essas alterações, o PHP começará a registrar erros no arquivo especificado (php_errors.log). 
Isso inclui erros de execução, problemas de conexão com o banco de dados e outras falhas.

Acompanhe o arquivo php_errors.log para monitorar e diagnosticar quaisquer problemas que ocorram em seu projeto.
